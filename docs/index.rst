YouTube to Notes
================

Transform YouTube videos into structured, editorial-style notes with AI-powered
analysis and intelligent screenshot extraction.

YouTube to Notes uses LLMs to analyze video transcripts and generate
comprehensive markdown notes with embedded screenshots at key moments. It
supports multiple AI providers (Gemini, OpenAI, Anthropic), parallel chunk
processing for long videos, and exports to PDF and Notion.
